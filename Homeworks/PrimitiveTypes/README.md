# Primitive Types

### Number Types

- Infinity, -Infinity
- NaN

### operators

... + , - , \* , / , %

```javascript
const a = 2;
const b = 10;
console.log(a + b);
```

### String Type

```javascript
const a = "Java";
const b = "Script";
console.log(a + b); // "Javascript"
```

### Boolean Type

- true
- false

```javascript
let a = 3;
let b = 4;
console.log(a > b); // false
```
